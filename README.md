# 🎬 Galaxy Filmes - Sistema de Aluguel

Este é um projeto prático desenvolvido como parte do curso **Técnico em Informática - Desenvolvimento Web**, focado na gestão e organização de um sistema de aluguel de filmes.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação das páginas web.
* **CSS3**: Estilização e layout (design responsivo).
* **JavaScript**: Lógica principal da aplicação (front-end e back-end).
* **Node.js**: Ambiente de execução para o servidor.
* **SQLite/Local DB**: Gerenciamento de dados (arquivos `.db`).
* **Markdown**: Para documentação do projeto.

## 📁 Estrutura do Projeto

* `public/`: Contém os arquivos de interface visíveis ao usuário (HTML, CSS, imagens).
* `server.js`: Arquivo principal que inicia e configura o servidor Node.js.
* `package.json`: Gerenciamento de dependências e scripts do projeto.
* `alunos.db` / `dados.db`: Bancos de dados onde as informações do sistema são armazenadas.
* `.gitignore`: Arquivo para instruir o Git sobre quais arquivos ou pastas não devem ser enviados ao repositório.
